中值定理自测

点击任意条目展开/收起；也可用下方按钮一次性展开或收起全部。
最值定理（极值存在定理 / Weierstrass）
闭区间上连续 ⇒ 在区间内必有最大值与最小值。
\[ \text{若 } f \text{ 在 } [a,b] \text{ 上连续，}\ \exists\, x_{\min},x_{\max}\in[a,b],\ f(x_{\min})\le f(x)\le f(x_{\max}),\ \forall x\in[a,b]. \]
介值定理（Intermediate Value Theorem）
闭区间上连续、两端点函数值给定 ⇒ 端点值之间的任何值都能取到。
\[ \text{若 } f \text{ 在 } [a,b] \text{ 上连续，}\ \forall\, y_0 \text{ 介于 } f(a)\text{ 与 }f(b)\text{ 之间},\ \exists\, c\in[a,b],\ f(c)=y_0. \]
零点定理（Bolzano 定理）
闭区间上连续且端点异号 ⇒ 区间内至少有一个零点。
\[ f \text{ 在 }[a,b]\text{ 上连续，且 } f(a)\cdot f(b)<0 \ \Rightarrow\ \exists\, c\in(a,b),\ f(c)=0. \]
费马中值定理（Fermat）
可导函数在（区间内部）某点取得极值 ⇒ 该点导数为 0。
\[ \text{若 } f \text{ 在 } c \text{ 处取极值且可导，}\ \Rightarrow\ f'(c)=0. \]
罗尔中值定理（Rolle）
闭区间连续、开区间可导，且端点函数值相等 ⇒ 存在一点导数为 0。
\[ f(a)=f(b) \ \Rightarrow\ \exists\, c\in(a,b),\ f'(c)=0. \]
平均值定理
闭区间连续、开区间可导 ⇒ 存在一点切线斜率等于割线斜率。
\[ f \text{ 在 }[a,b]\text{ 上连续，}(a,b)\text{ 上可导} \ \Rightarrow\ \exists\, c\in(a,b),\ f'(c)=\frac{f(b)-f(a)}{\,b-a\,}. \]
拉格朗日中值定理（平均斜率）
\[ \exists\, c\in(a,b),\quad f'(c)=\frac{f(b)-f(a)}{\,b-a\,}. \]
柯西中值定理（Cauchy MVT）
两函数的拉格朗日推广：导数比值 = 平均增量比值（要求 \(g'(x)\neq0\)）。
\[ f,g \text{ 在 }[a,b]\text{ 上连续，}(a,b)\text{ 上可导，且 }g'(x)\neq 0\ \Rightarrow\ \exists\, c\in(a,b),\ \frac{f'(c)}{g'(c)}=\frac{f(b)-f(a)}{\,g(b)-g(a)\,}. \]
泰勒公式（带拉格朗日余项）
\[ f(x)= f(a)+f'(a)(x-a)+\frac{f''(a)}{2!}(x-a)^2+\cdots+\frac{f^{(n)}(a)}{n!}(x-a)^n +\frac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1},\;\;\xi\in(a,x). \]
积分中值定理（由拉格朗日中值定理推出）
闭区间连续 ⇒ 面积等于某点函数值乘以区间长度。
\[ \text{若 } f \text{ 在 }[a,b]\text{ 上连续，}\ \exists\, c\in(a,b),\ \int_a^b f(x)\,dx = f(c)\,(b-a). \]
